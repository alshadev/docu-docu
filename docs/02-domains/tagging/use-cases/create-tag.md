---\ntitle: "Create Tag"\nsidebar_label: "Create Tag"\n---\n\n# Create Tag\n\n## Overview\nThis command creates a new tag.\n\n## Request Payload\n\n| Field | Type | Required | Description |\n|---|---|---|---|\n| name | string | Yes | Name of the tag |\n\n## Response Payload\n\n| Field | Type | Description |\n|---|---|---|\n| id | UUID | ID of the newly created tag |\n| name | string | Name of the tag |\n\n## Business Rules\n\n1.  Tag names must be unique.\n\n## Validation Rules\n\n1.  Name must be present.\n2.  Name must be a string.\n\n## Workflow\n\n1.  Validate the request payload.\n2.  Create a new tag with the given name.\n3.  Return the newly created tag.\n\n## Error Scenarios\n\n| Scenario | HTTP Status | Error Code |\n|---|---|---|\n| Tag name already exists | 400 | TAG\_NAME\_EXISTS |\n
