---\ntitle: "Create Tag"\nsidebar_label: "Create Tag"\n---\n\n# Create Tag\n\n## Overview\nCreates a new tag.\n\n## Request Payload\n\n| Field | Type | Required | Description |\n|-------|------|----------|-------------|\n| name | string | Yes | Name of the tag. |\n\n## Response Payload\n\n| Field | Type | Description |\n|-------|------|-------------|\n| id | string | Unique identifier of the created tag. |\n| name | string | Name of the tag. |\n\n## Business Rules\n\n1.  Tag name must be unique.\n\n## Validation Rules\n\n1.  Tag name must be between 3 and 50 characters.\n\n## Workflow\n\n1.  Receive request to create a tag with a name.\n2.  Validate the tag name.\n3.  Create a new tag with the given name.\n4.  Return the newly created tag.\n\n## Error Scenarios\n\n| Scenario | HTTP Status | Error Code |\n|----------|-------------|------------|\n| Tag name is not unique | 400 | TAG\_NAME\_DUPLICATE |\n| Invalid tag name | 400 | INVALID\_TAG\_NAME |\n
