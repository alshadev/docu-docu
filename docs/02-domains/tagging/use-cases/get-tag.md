---\ntitle: "Get Tag"\nsidebar_label: "Get Tag"\n---\n\n# Get Tag\n\n## Overview\nThis query retrieves a tag by its ID.\n\n## Request Payload\n\n| Field | Type | Required | Description |\n|---|---|---|---|\n| id | UUID | Yes | ID of the tag to retrieve |\n\n## Response Payload\n\n| Field | Type | Description |\n|---|---|---|\n| id | UUID | ID of the tag |\n| name | string | Name of the tag |\n\n## Business Rules\n\nN/A\n\n## Validation Rules\n\n1.  ID must be a valid UUID.\n\n## Workflow\n\n1.  Validate the request payload.\n2.  Retrieve the tag with the given ID.\n3.  Return the tag.\n\n## Error Scenarios\n\n| Scenario | HTTP Status | Error Code |\n|---|---|---|\n| Tag not found | 404 | TAG\_NOT\_FOUND |\n
