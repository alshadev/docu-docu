---\ntitle: "Delete Tag"\nsidebar_label: "Delete Tag"\n---\n\n# Delete Tag\n\n## Overview\nThis command deletes an existing tag.\n\n## Request Payload\n\n| Field | Type | Required | Description |\n|---|---|---|---|\n| id | UUID | Yes | ID of the tag to delete |\n\n## Response Payload\n\nNo content.\n\n## Business Rules\n\n1.  Tag cannot be deleted if it is associated with any products.\n\n## Validation Rules\n\n1.  ID must be a valid UUID.\n\n## Workflow\n\n1.  Validate the request payload.\n2.  Delete the tag with the given ID.\n\n## Error Scenarios\n\n| Scenario | HTTP Status | Error Code |\n|---|---|---|\n| Tag not found | 404 | TAG\_NOT\_FOUND |\n| Tag is associated with products | 400 | TAG\_IN\_USE |\n
