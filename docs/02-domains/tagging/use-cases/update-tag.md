---\ntitle: "Update Tag"\nsidebar_label: "Update Tag"\n---\n\n# Update Tag\n\n## Overview\nThis command updates an existing tag.\n\n## Request Payload\n\n| Field | Type | Required | Description |\n|---|---|---|---|\n| id | UUID | Yes | ID of the tag to update |\n| name | string | No | New name of the tag |\n\n## Response Payload\n\n| Field | Type | Description |\n|---|---|---|\n| id | UUID | ID of the updated tag |\n| name | string | Name of the tag |\n\n## Business Rules\n\n1.  Tag names must be unique.\n\n## Validation Rules\n\n1.  ID must be a valid UUID.\n2.  Name must be present if provided.\n3.  Name must be a string.\n\n## Workflow\n\n1.  Validate the request payload.\n2.  Update the tag with the given ID.\n3.  Return the updated tag.\n\n## Error Scenarios\n\n| Scenario | HTTP Status | Error Code |\n|---|---|---|\n| Tag not found | 404 | TAG\_NOT\_FOUND |\n| Tag name already exists | 400 | TAG\_NAME\_EXISTS |\n
