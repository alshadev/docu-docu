---\ntitle: "Filter Products by Tags"\nsidebar_label: "Filter Products by Tags"\n---\n\n# Filter Products by Tags\n\n## Overview\nRetrieves a list of products filtered by one or more tags.  Products must have *all* of the specified tags to be included in the results.\n\n## Request Payload\n\n| Field | Type | Required | Description |\n|-------|------|----------|-------------|\n| tagIds | array[string] | Yes | List of tag IDs to filter products by. |\n\n## Response Payload\n\n| Field | Type | Description |\n|-------|------|-------------|\n| products | array[Product] | List of products that match all specified tags. |\n\n## Business Rules\n\n1.  Products must have all the specified tags to be included in the results.\n\n## Validation Rules\n\n1.  Each tag ID in the `tagIds` array must be a valid UUID and exist in the `Tag` entity.\n\n## Workflow\n\n1.  Receive request to filter products by a list of tag IDs.\n2.  Validate the tag IDs.\n3.  Retrieve products from the database that have all the specified tag IDs.\n4.  Return the list of filtered products.\n\n## Error Scenarios\n\n| Scenario | HTTP Status | Error Code |\n|----------|-------------|------------|\n| Invalid tag ID | 400 | INVALID\_TAG\_ID |\n
